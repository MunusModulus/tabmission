About

Tabmission is a Chrome extension designed to make tab handling and browsing navigation faster and smoother.

It focuses on reducing friction in daily browsing with lightweight, practical tools for people who work with many tabs.

Features

Fast Tab Search

Quickly find and switch tabs with a fast overlay search interface.

Link Selector

Select and open multiple links efficiently with a visual selection workflow.

PageJump (PJ)

Move through paginated pages more smoothly with practical URL-based navigation handling.

LastTab Switching

Return to the previously used tab quickly with optimized switching behavior.
