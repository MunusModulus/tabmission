Tabmission Privacy Policy

Last updated: April 22, 2026

Tabmission is a browser extension that provides tab control, tab search, search-box focus, page jump, and link selection features.

1. Data Collection and Transmission

Tabmission does not collect or transmit personal data to external servers.

Tabmission does not send user data to external services for advertising, analytics, profiling, or marketing purposes.

2. Data Processed Locally by the Extension

To provide its features, Tabmission may access and process the following information locally in the user's browser:

  • Tab titles, URLs, and tab state for tab search and tab control features
  • DOM elements on the current webpage for search-box focus, page jump, and link selection features
  • User settings such as shortcut keys, tab behavior, and saved site rules

This information is used locally only to operate the extension's features.

3. Stored Information

Tabmission stores user settings in Chrome extension storage.

  • Shortcut key settings
  • Tab behavior settings
  • Page jump settings and saved site rules
  • Search-box focus settings
  • Other settings required for the extension to operate properly

4. Third-Party Sharing

Tabmission does not sell, share, or provide user data to third parties.

5. Remote Code

Tabmission does not download or execute remotely hosted code. It uses only code bundled in the published extension package.

6. Changes to This Policy

This Privacy Policy may be updated if the extension's features or behavior change. Any updates will be posted on this page.

7. Contact

For questions about Tabmission, please use the support page or repository below.
